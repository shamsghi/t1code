Run instantly:

```bash
bunx @maria_rcks/t1code
```

Install globally:

```bash
bun add -g @maria_rcks/t1code
```

Develop from source:

```bash
git clone https://github.com/maria-rcks/t1code.git
cd t1code
bun install
bun dev:tui
```
